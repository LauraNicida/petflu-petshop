# PetFlu Petshop — Tema Tricolor (sem audiovisual)

Site temático para cães tricolores, mantendo o escopo acadêmico:
- 3 categorias de produtos (2+ itens cada).
- Serviços: banho e tosa, com opção de tele-busca.
- Visual em verde, grená e branco (sem uso de escudos/logotipos oficiais).

## Como usar
Abra `index.html` localmente ou publique no GitHub Pages.

> Observação: imagens usam links do Unsplash (livre para uso). Você pode trocá-las depois.
